# Python Tools (Cybersecurity Toolkit)

This section contains Python-based cybersecurity tools developed during my learning journey. The projects focus on automation, reconnaissance, log analysis, and security validation tasks that I used in penetration testing and system analysis.

---

## I) Port Scanner

A simple port scanning tool used to identify open ports on a target system. It helps in understanding exposed services and potential attack surfaces during reconnaissance.

File: `port-scanner.py`

---

## II) Authentication Log Analyzer

A script that analyzes Linux authentication logs (`auth.log`) to detect failed login attempts and suspicious activity patterns.

File: `auth-log-analyzer.py`

---

## III) Password Strength Checker

A tool used to evaluate the strength of passwords based on common security rules such as length, character variety, and complexity.

File: `password_checker.py`

---

## IV) Simple Log Analyzer

A lightweight script used to scan and analyze log files for specific keywords and repeated events, helping in basic forensic analysis.

File: `simple_log_analyser.py`

---

## V) IP Geolocation Lookup Tool

A script that retrieves geolocation and network information for a given IP address using an external API. It is useful in OSINT and reconnaissance activities.

File: `ip-geolocation-lookup.py`

---

## VI) Banner Grabbing


- Port scanning of common services
- Detection of open ports
- Banner grabbing for service identification
- Error handling for stable scanning

  File: `port_scanner_banner_grab.py`
