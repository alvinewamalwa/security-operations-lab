## 🔍 1. Cybersecurity Search Skills

### Tools & Concepts Learned:
- CVE (Common Vulnerabilities and Exposures): A database of known security vulnerabilities.
- VirusTotal: A tool used to analyze files, URLs, and detect malicious activity.
- Security research mindset: Understanding how attackers and defenders use public security intelligence.

### Key takeaway:
Security professionals rely heavily on threat intelligence tools to investigate risks and vulnerabilities.

## 🐧 2. Linux Fundamentals (Part 1)

### Concepts Covered:
- Linux is an open-source operating system widely used in servers and cybersecurity environments.
- Understanding file system structure (directories and paths).

### Commands Learned:
- `ls` → list files and directories  
- `cd` → change directory  
- `pwd` → show current directory path  
- `mkdir` → create directories  
- `touch` → create files  
- `find` → search for files and directories  
- `grep` → search inside files
- etc

### Shell Concept:
- The shell is the interface used to interact with the Linux system.
- Commands are executed through the shell to manage the system.

- Cybersecurity involves both technical system knowledge and intelligence tools.
- Linux is a core skill for cybersecurity roles because most servers run on Linux.
- Search skills are essential for investigating threats and vulnerabilities.
