# Log Analysis Lab

## Objective
To analyze system logs and identify suspicious patterns.

## Tools Used
- grep
- cat
- less

## Process
- Accessed system log files
- Searched for keywords like "failed", "error", "unauthorized"
- Filtered logs using `grep`
- Identified repeated failed login attempts

## Observations / Findings
- Multiple failed login attempts detected from unknown IPs
- Error logs indicate repeated authentication failures

## Security Insight
Log analysis helps detect brute force attacks and suspicious system activity early.
